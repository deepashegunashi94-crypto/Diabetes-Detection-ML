# Diabetes Prediction using Machine Learning

## Overview
This project predicts whether a person is diabetic or non-diabetic using Machine Learning. It is built using Python and Logistic Regression with a custom diabetes dataset. Users can provide medical details, and the model predicts the diabetes status.

## Features
- Diabetes prediction using Logistic Regression
- Data preprocessing and model training
- User input for prediction
- Accuracy evaluation
- Simple and easy-to-understand implementation

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- CSV Dataset

## How It Works
1. Load the diabetes dataset from a CSV file.
2. Select important medical features.
3. Split the dataset into training and testing sets.
4. Train the Logistic Regression model.
5. Accept user input for medical parameters.
6. Predict whether the person is diabetic or non-diabetic.

## Input Features
- Age
- Glucose
- BMI
- Blood Pressure
- Insulin

## Output
- Diabetic
- Non-Diabetic

## Applications
- Early diabetes risk prediction
- Healthcare support systems
- Medical data analysis
- Educational Machine Learning project
